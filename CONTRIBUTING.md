# Contributing

This repository is a personal portfolio. If you spot a typo, broken link, or accessibility issue:

1. Open an issue describing the problem.
2. If you prefer to submit a fix:
   - Fork the repo and create a branch.
   - Make small, focused changes (e.g. one fix per PR).
   - Ensure HTML/CSS/JS remains valid and that the site still works at 375px, 768px, and 1024px widths.
   - Submit a pull request with a short description.

No formal style guide beyond the existing code: keep markup semantic, CSS organized by section/component, and JS modular and commented.

Thanks for your interest.
